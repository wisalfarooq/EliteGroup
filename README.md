# Elite Group - Full Project

This is a full-stack starter for Elite Group (React + Tailwind frontend, Node/Express backend with MongoDB).

## Running

1. Backend:
   - cd backend
   - npm install
   - copy .env.example to .env and fill MONGO_URI and JWT_SECRET
   - npm run seed
   - npm run dev

2. Frontend:
   - cd frontend
   - npm install
   - npm run dev
